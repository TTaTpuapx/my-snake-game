# 🐍 My Snake Game

Классическая игра «Змейка» с минималистичным дизайном и плавным управлением.  
Цель — вырастить самую длинную змею, не врезавшись в стены или собственный хвост.

## ✨ Возможности

- Классический геймплей с растущей змейкой и случайным появлением еды
- Простое и отзывчивое управление (стрелки или WASD)
- Отслеживание текущего счёта и рекорда (сохраняется между сессиями)
- Возможность поставить игру на паузу
- Адаптивная скорость: чем длиннее змея, тем быстрее она движется
- Чистый и легко читаемый код (отличная основа для изучения Pygame)
