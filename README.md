\MySnakeGame



Учебный проект по написанию простой 2D игры змейка.  

Используемые технологии: Java SE, Swing.



\Управление

\- W — вверх

\- S — вниз

\- A — влево

\- D — вправо



\Особенности игры

\- Змейка ест яблоки (красного цвета) и растет

\- При достижении края игрового поля появляется на противоположной стороне

\- Змейка может столкнуться с собой

\- Игра бесконечна
